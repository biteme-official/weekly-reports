사용 방법

보고서 작성 (팀원)

  1. 레포를 clone합니다
    git clone https://github.com/biteme-official/weekly-reports.git
cd weekly-reports
  2. 자신의 프로젝트 폴더에서 Claude Code를 실행합니다
    claude
  3. 보고서를 생성합니다
    /weekly-report          # 지난 주 주간 보고서
/weekly-report 6월      # 6월 월간 보고서
/weekly-report W25      # 특정 주차
/weekly-report 3개월    # 최근 3개월
  4. 생성된 파일을 확인하고 push합니다
    git add .
git commit -m "W28 주간 보고서 제출"
git push

파일 위치 규칙

reports/2026/W28/이름.md주간 보고서 (ISO 주차)
reports/2026/M07/이름.md월간 보고서
reports/2026/W28/SUMMARY.md관리자 취합본

보고서 확인 (관리자)

이 뷰어에서 바로 확인 가능합니다.

전체/주간/월간 탭으로 필터링하고, 기간을 클릭하면 해당 기간 전체 보고서가 표시됩니다. 이름을 클릭하면 개인 보고서만 볼 수 있습니다.

취합본 생성 (관리자)

이 레포 폴더에서 Claude Code를 실행한 후:

취합본 생성해줘  # 또는 SUMMARY 만들어줘

해당 주차의 모든 보고서를 읽고 SUMMARY.md를 자동 생성합니다.

사전 준비

Claude Code가 설치되어 있어야 합니다:

npm install -g @anthropic-ai/claude-code
